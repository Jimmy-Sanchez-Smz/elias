- Introducción -

El HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar páginas web. Funciona mediante el uso de etiquetas que definen el contenido y la organización de elementos como texto, imágenes, enlaces, tablas y formularios. HTML es la base fundamental de la web, y su simplicidad permite que tanto principiantes como desarrolladores avanzados puedan construir sitios y aplicaciones web.

Utilizar código HTML implica comprender cómo funcionan sus etiquetas y atributos para estructurar documentos de manera eficiente. Combinado con tecnologías como CSS (para estilos) y JavaScript (para interactividad), HTML permite crear experiencias web dinámicas y atractivas. Aprender a usar HTML es el primer paso para dominar el desarrollo web y comprender la arquitectura básica de Internet.

- Resultados -

Se logró crear con éxito una página web simple. La página principal (index.html) con títulos, texto e imagen y una página secundaria (pagina2.html) enlazada desde la página principal. Ambas páginas se visualizan correctamente en el navegador web y sus contenidos se cargaron exitosamente en GitHub Pages y son accesibles en línea.

  • Estetica: Se debe de ver bien, debe de ser atractica a simple vista y ordenado
  • Efectividad: La pagina no debria de tener ningun tipo de problema a la hora de cargar y debe de cumplir con lo que se solicita
  • Accesibilidad: La pagina debe de ser accesible al navegar y utilizar
